# Docker Java Spring Boot

## Maven 3 Spring Boot 3 - JDK 17
###
```bash
docker build . -f Dockerfile.mvn.17 -t jakharbek/java-spring-boot-mvn-17
```

```bash
docker run -it jakharbek/java-spring-boot-mvn-17 sh
```
